本机 Ollama 安装配置（推荐）

① 下载安装 前往 ollama.com 下载对应系统的安装包（Windows / macOS / Linux），按向导安装完成后 Ollama 会作为后台服务自动运行。

② 启动服务 安装后一般自动运行；如未启动，在终端执行 ollama serve（监听 127.0.0.1:11434）。

③ 拉取模型 在终端执行以下任意一条命令（首次需要下载模型文件）：

ollama pull qwen2.5:7b　　中文优先，约 4.7 GB，推荐日常使用

ollama pull qwen2.5:14b　理解更强，需 8 GB+ 显存/内存

ollama pull llama3.2:3b　　英文轻量，约 2.0 GB，低配可用

④ 验证连接 模型下载完成后，刷新本页面或点击"⟳"按钮——AI 助手状态栏变为绿色即表示连接成功。

如在局域网服务器部署 Ollama，请在下方输入服务器地址；本机不可用时自动回退到该地址。
